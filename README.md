# generator-webpack
> webpack4

## Installation
install`yo`，update`submodule`
```bash
npm install -g yo
npm i
```

Then generate your new project in local:

```bash
npm link
```

Make new dir

```bash
mkdir test-yo
cd test-yo
yo webpack
```

Add git hooks

```bash
npm run hooks
```
